# pages

Cada archivo es una página completa de la app.
Usan los componentes de ui/ y layout/ para construirse.

Páginas previstas:
- HomePage.jsx       → portada con álbumes destacados y actividad reciente
- ExplorePage.jsx    → grid de álbumes con buscador y filtro por género
- AlbumPage.jsx      → detalle de un álbum: info, reseñas, favorito
- ProfilePage.jsx    → perfil de usuario: sus reseñas y favoritos
- LoginPage.jsx      → formulario de login y registro
- RankingsPage.jsx   → rankings y estadísticas generales
