# components/ui

Componentes pequeños y reutilizables que no tienen lógica de negocio.
Se usan en toda la app.

Ejemplos:
- AlbumCard.jsx      → tarjeta de álbum con portada, título y puntuación
- Button.jsx         → botón primario y secundario
- StarRating.jsx     → estrellas de puntuación del 1 al 5
- Badge.jsx          → etiqueta de género (ej: "Hip-hop")
- Avatar.jsx         → foto de perfil de usuario
